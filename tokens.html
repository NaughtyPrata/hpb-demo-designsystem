Implementation Guide

How to integrate HPB Design Tokens into your development workflow.

CSS Implementation

1. Import CSS Variables

@import url('hpb-tokens.css');

/* Use tokens in your styles */
.button-primary {
  background-color: var(--hpb-primary-500);
  color: white;
  padding: var(--spacing-md);
  border-radius: var(--radius-md);
}

2. Component Styling

.health-card {
  background: white;
  border: 1px solid var(--hpb-neutral-200);
  border-radius: var(--radius-lg);
  padding: var(--spacing-lg);
  box-shadow: var(--shadow-sm);
}

JavaScript Implementation

1. Import JSON Tokens

import tokens from './hpb-tokens.json';

// Access color tokens
const primaryColor = tokens.colors.primary[500];
const spacing = tokens.spacing.md;

2. React/Vue Usage

// React component
const HealthCard = ({ children }) => (
  <div style={{
    backgroundColor: 'white',
    border: `1px solid ${tokens.colors.neutral[200]}`,
    borderRadius: tokens.radius.lg,
    padding: tokens.spacing.lg
  }}>
    {children}
  </div>
);

Tailwind CSS Configuration

tailwind.config.js

module.exports = {
  theme: {
    extend: {
      colors: {
        'hpb-primary': {
          50: '#f0fdf4',
          100: '#dcfce7',
          500: '#22c55e',
          900: '#14532d'
        }
      },
      fontFamily: {
        'primary': ['Inter', 'sans-serif']
      },
      spacing: {
        'xs': '0.25rem',
        'sm': '0.5rem',
        'md': '1rem'
      }
    }
  }
}
